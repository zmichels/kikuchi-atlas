# KIKU-T043: Build the forsterite spherical dictionary fixture

## Description

Use a coarse cube-shell directional lattice to publish a compact
orientation-dictionary fixture. Preserve the source-field hash and make the
three supported quarter-turn orientations and validation result explicit.

## Acceptance Criteria

- [x] The fixture is emitted from the declared local S2 source after exact source-hash validation.
- [x] The source is downsampled by declared nearest-direction sampling only; no unspecified interpolation or filtering is introduced.
- [x] A consumer can reproduce the included validation ranking using the documented crystal-to-sample convention.
- [x] The local package is built and verified without adding generated bytes to Git.
