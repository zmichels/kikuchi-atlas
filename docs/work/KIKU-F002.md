# KIKU-F002: Kikuchipy kinematical reference products

## Description

Expose kikuchipy's existing stereographic, spherical, Lambert-master, and
detector-projected kinematical capabilities as deterministic project products
before deciding whether a custom dynamical/kinematical hybrid is warranted.

## Acceptance Criteria

- [ ] All six implementation tasks have accepted evidence.
- [ ] A cited forsterite recipe reproduces the selected orientation in every projection.
- [ ] Reflection selection and coordinate conventions are explicit and inspectable.
- [ ] Native-scale figures establish a durable visual decision gate for later hybrid work.
