# KIKU-T028: Publish shared Ice Ih art band catalog

## Description

Ice Ih axial bands, records globe and tattoo eligibility, and partitions globe
members into four nonempty tie-aware strength cohorts.

- [ ] A strict catalog snapshot and bounded bundle reject forged or incomplete content before atomic publication.
- [ ] The real Ice catalog records all source, recipe, threshold, member, cohort, and claim-boundary evidence.
